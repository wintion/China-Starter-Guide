## Réponse rapide

Vous n’avez pas besoin de vingt applications pour voyager en Chine.  
Pour la plupart des visiteurs, une base solide suffit :

- une application de paiement ;
- une application de messagerie ;
- une solution de VTC ;
- un ou deux outils de cartographie ;
- une application de traduction.

Le vrai problème n’est pas d’en avoir trop peu.  
Le vrai problème, c’est d’arriver avec les mauvaises applications et aucune configuration prête.

---

## Le noyau d’applications à préparer

### 1. Alipay

Pour beaucoup de visiteurs étrangers, c’est l’application tout-en-un la plus utile.

Elle sert souvent à :

- payer au quotidien ;
- accéder à des mini-programmes ;
- commander un Didi ;
- utiliser certaines fonctions de transport urbain ;
- gérer de petits outils pratiques pendant le voyage.

Si vous ne deviez installer qu’une seule application vraiment essentielle avant le départ, **Alipay** serait souvent le meilleur choix.

### 2. WeChat

**WeChat** compte beaucoup parce que :

- c’est l’application de messagerie par défaut en Chine ;
- les entreprises et les contacts locaux l’utilisent souvent ;
- certains services passent par ses mini-programmes ;
- elle peut aussi servir de solution de paiement de secours si elle est bien configurée.

Vous n’avez pas besoin d’en maîtriser toutes les fonctions.  
Vous avez surtout besoin d’un compte prêt à fonctionner.

### 3. Une configuration cartographique réaliste

Il n’existe pas une seule application de cartes parfaite pour tous les voyageurs étrangers.

Une combinaison raisonnable ressemble souvent à ceci :

- **Apple Maps** pour une expérience plus accessible en anglais sur iPhone ;
- **Amap (Gaode)** ou **Baidu Maps** pour un niveau de détail local plus fort, si vous acceptez une interface plus chinoise.

Le bon montage consiste souvent à combiner une carte plus lisible côté anglais avec une carte plus solide côté données locales.

### 4. Didi

**Didi** est l’outil de VTC que la plupart des voyageurs finissent par utiliser.

Vous pouvez y accéder :

- via l’application autonome ;
- dans Alipay ;
- dans WeChat.

Ce qui compte, ce n’est pas la porte d’entrée.  
Ce qui compte, c’est de pouvoir commander une voiture vite et proprement quand il le faut.

### 5. Une application de traduction

Votre application de traduction n’est pas là pour vous rendre fluent.  
Elle sert surtout pour :

- les adresses ;
- les menus ;
- les échanges au comptoir ;
- les panneaux ;
- les petits moments de dépannage.

La traduction de texte et d’image compte bien plus que le fantasme de la conversation instantanée parfaite.

---

## Les bonnes applications secondaires

Selon votre voyage, d’autres applications peuvent aussi aider :

- **Trip.com** pour gérer transports et hôtels ;
- un outil de réservation ferroviaire si vous prenez beaucoup le train ;
- les applications des compagnies aériennes pour les vols intérieurs ;
- des packs de traduction hors ligne et des notes enregistrées.

Ajoutez-les seulement si vous savez précisément pourquoi vous en avez besoin.

---

## Les étapes de configuration qui comptent avant l’arrivée

- connectez-vous avant de prendre l’avion ;
- liez votre moyen de paiement le plus tôt possible ;
- vérifiez que l’application s’ouvre bien et reste connectée ;
- enregistrez en chinois les adresses importantes dans vos notes ou captures d’écran ;
- évitez de faire votre toute première configuration à l’aéroport en plein décalage horaire.

La préparation compte bien davantage que le nombre total d’applications.

---

## Les erreurs les plus courantes

- installer trop d’applications chinoises sans en comprendre vraiment aucune ;
- supposer que WeChat suffit à tout faire ;
- remettre la configuration du paiement à après l’arrivée ;
- dépendre d’une seule application de cartographie ;
- négliger les notes et captures d’écran hors ligne.

La meilleure configuration est simple, peu spectaculaire et déjà testée.

---

## En pratique

- En Chine, une seule application regroupe souvent plusieurs services.
- L’anglais existe, mais de manière inégale.
- Les menus et les fonctions changent avec le temps.
- Le bon ensemble d’applications dépend aussi un peu de votre téléphone, de votre configuration de paiement et de votre style de voyage.

Vous n’êtes pas en train de recréer toute une vie numérique.  
Vous êtes simplement en train de monter une boîte à outils fiable pour le voyage.

---

## Liste de contrôle

- [ ] Alipay est installé et s’ouvre correctement.
- [ ] WeChat est installé et connecté.
- [ ] Je peux accéder à Didi d’une manière ou d’une autre.
- [ ] J’ai une configuration cartographique que je comprends vraiment.
- [ ] Mon application de traduction et mes notes hors ligne sont prêtes.
