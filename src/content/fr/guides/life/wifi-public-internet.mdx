## Réponse rapide

Le Wi-Fi public existe en Chine, mais il constitue une base fragile pour voyager.

Il faut le voir comme :

- un bonus ;
- une solution de secours ;
- un outil pour les tâches non urgentes.

Il ne faut pas construire tout le voyage dessus pour :

- les paiements ;
- la navigation en mouvement ;
- les codes urgents ;
- la récupération de compte ;
- les réservations à la dernière minute.

---

## Pourquoi les voyageurs se font piéger

Le problème n'est pas que le Wi-Fi public existe.  
Le problème est d'attendre qu'il se comporte comme votre connexion domestique ou comme le Wi-Fi d'un aéroport très bien équipé ailleurs.

Les points de friction fréquents sont :

- les pages de connexion ;
- les vérifications d'identité ;
- des performances faibles ;
- un accès irrégulier aux services dont vous avez vraiment besoin.

"Connecté" ne veut pas toujours dire "utile".

---

## Où vous verrez généralement du Wi-Fi

Vous en trouverez souvent dans :

- les hôtels ;
- les aéroports ;
- les cafés ;
- les centres commerciaux ;
- certains trains à grande vitesse.

Mais cela ne veut pas dire que ce sera la bonne connexion pour les moments importants.

---

## Le meilleur réflexe

Considérez les **données mobiles** comme votre connexion principale.  
Considérez le Wi-Fi comme secondaire.

Ce simple changement d'approche enlève déjà beaucoup de stress inutile.

---

## Ce à quoi le Wi-Fi sert bien

Le Wi-Fi public est raisonnablement utile pour :

- naviguer sans urgence ;
- télécharger une mise à jour non critique ;
- synchroniser des photos plus tard ;
- regarder ou lire quelque chose qui peut attendre.

Si un échec peut réellement vous bloquer sur le moment, ne comptez pas sur lui.

---

## Ce à quoi il ne faut pas trop se fier

Évitez d'en dépendre pour :

- un paiement à la caisse ;
- commander un VTC dans la rue ;
- recevoir un code de vérification urgent ;
- réinitialiser un mot de passe ;
- toute action qui doit marcher vite pendant que vous vous déplacez.

L'Internet public est précisément l'endroit où les configurations numériques fragiles se révèlent.

---

## Le Wi-Fi de l'hôtel est souvent meilleur, mais pas magique

Le Wi-Fi d'hôtel est souvent plus utile que les réseaux publics aléatoires, mais il reste très variable.

Il peut être très bien pour :

- organiser la soirée ;
- faire une session de navigation plus longue ;
- régler calmement des tâches secondaires.

Ce n'est toujours pas quelque chose à qui confier aveuglément les moments critiques.

---

## En pratique

- Le Wi-Fi peut aider, mais il ne doit pas être l'épine dorsale du voyage.
- Les données mobiles restent généralement la solution la plus pratique.
- Les sauvegardes hors ligne sont importantes parce que toute connexion peut échouer.
- Les voyageurs qui pensent "le Wi-Fi me sauvera" sont souvent ceux qui perdent le plus de temps.

Planifiez comme s'il pouvait vous décevoir, et tout ira beaucoup mieux.

---

## Liste de contrôle

- [ ] Je considère les données mobiles comme ma connexion principale.
- [ ] J'ai enregistré hors ligne les adresses et réservations importantes.
- [ ] J'utiliserai le Wi-Fi public seulement pour des tâches non critiques.
- [ ] Je ne compterai pas sur lui pour payer ou vérifier un compte.
- [ ] J'ai un plan de secours si la connexion en direct devient inutilisable.
