## Réponse rapide

Oui, les étrangers peuvent utiliser Alipay en Chine, et pour beaucoup de visiteurs c’est même le portefeuille le plus simple pour commencer.

Mais la version réaliste est la suivante :

- Alipay est très utile ;
- Alipay n’est pas universel ;
- Alipay a quand même besoin d’un secours.

Vous n’êtes vraiment prêt qu’après avoir :

- créé le compte ;
- lié une carte ;
- terminé les vérifications demandées ;
- et réussi un vrai paiement.

---

## Pourquoi Alipay paraît souvent le plus simple au début

Pour beaucoup de voyageurs, Alipay paraît plus direct que WeChat au départ parce qu’il ressemble davantage à un outil de voyage qu’à une application sociale à laquelle on a ajouté du paiement.

Il sert souvent pour :

- les paiements par QR code en magasin ;
- les taxis et VTC ;
- certaines fonctions de transport ;
- les billets et réservations ;
- plusieurs outils pratiques intégrés à l’application.

Les guides officiels d’Alipay pour les voyageurs internationaux confirment d’ailleurs que les utilisateurs étrangers peuvent s’inscrire, lier des cartes internationales et payer leurs dépenses courantes en Chine continentale.

---

## Ce qu’un Alipay lié à une carte étrangère permet réellement

Pour un voyageur de court séjour, Alipay est avant tout un **outil de paiement du quotidien**.

En pratique, cela signifie qu’il fonctionne surtout bien pour :

- les repas et boissons ;
- les magasins et supermarchés ;
- les VTC ;
- les dépenses ordinaires de voyage.

En revanche, cela ne veut pas dire que vous obtenez exactement les mêmes fonctions financières qu’un résident disposant d’un compte bancaire local.

Les guides officiels d’Alipay précisent que certaines fonctions locales ne sont pas disponibles ou restent limitées pour les cartes étrangères, notamment :

- les transferts entre particuliers ;
- les enveloppes rouges ;
- les produits de gestion de patrimoine ;
- certains services financiers ou d’assurance.

Pour un voyageur, ce n’est généralement pas un problème.  
Le but est de payer dans la vraie vie, pas de reconstituer une vie bancaire locale complète.

---

## La meilleure façon de le mettre en place

### 1. Commencez par l’inscription

Si l’application propose un parcours plus clair pour utilisateurs internationaux, prenez-le.

### 2. Ajoutez la carte avec soin

Essayez de garder les informations cohérentes entre :

- le passeport ;
- la carte ;
- le compte.

Si la première carte échoue, inutile d’insister dix fois sur le même écran.  
Essayez plutôt une carte d’un autre émetteur.

### 3. Terminez la vérification si elle apparaît

La vérification n’est pas décorative.  
Elle joue sur les plafonds et sur la stabilité réelle des paiements.

### 4. Faites un petit test sans enjeu

Essayez Alipay dans une supérette, un café ou un supermarché.  
Le premier petit paiement réussi vaut plus qu’un simple message “carte ajoutée avec succès”.

---

## Les endroits où Alipay paraît souvent le plus fort

En pratique, Alipay est particulièrement confortable dans :

- les chaînes ;
- les supermarchés ;
- les centres commerciaux ;
- les commerces orientés voyageurs ;
- les réservations de services et les VTC.

Il est aussi souvent plus agréable pour les visiteurs qui aiment avoir un tableau de bord paiement + voyage, plutôt qu’une expérience centrée sur la messagerie.

---

## Pourquoi il faut quand même un plan de secours

Même si Alipay marche très bien la plupart du temps, gardez un secours pour :

- les très petits commerçants informels ;
- les codes de collecte personnels ;
- les commerçants qui acceptent Alipay mais pas votre route exacte de carte étrangère ;
- les problèmes de réseau ;
- les frictions du premier jour.

Un bon secours, c’est :

- WeChat Pay ;
- une autre carte étrangère ;
- un peu de RMB en liquide.

Si vous préparez uniquement Alipay sans rien d’autre, vous rendez votre voyage plus fragile qu’il ne devrait l’être.

---

## Les raisons les plus fréquentes d’un échec Alipay

- la carte est liée, mais la banque bloque la transaction réelle ;
- le commerçant accepte Alipay, mais pas votre route spécifique de carte étrangère ;
- la vérification du compte n’est pas terminée ;
- vous avez ouvert le mauvais flux QR ;
- le réseau ou l’appareil du caissier est temporairement instable.

Ces échecs sont suffisamment courants pour qu’il faille les traiter comme des incidents normaux de voyage, pas comme la preuve que toute la configuration est cassée.

---

## Que faire si Alipay échoue à la caisse

1. vérifiez si le paiement a vraiment échoué ou s’il est simplement en attente ;
2. demandez au commerçant s’il a reçu la confirmation ;
3. n’appuyez pas plusieurs fois à l’aveugle ;
4. passez une seule fois à un autre portefeuille, une autre carte ou au liquide ;
5. vérifiez l’historique plus tard, au calme.

À la caisse, la priorité n’est pas de comprendre la cause exacte.  
La priorité est d’avancer proprement.

---

## Liste de contrôle pratique

- [ ] J’ai réussi à ouvrir et utiliser mon compte Alipay.
- [ ] J’ai ajouté au moins une carte étrangère.
- [ ] J’ai terminé la vérification si elle a été demandée.
- [ ] Je sais où se trouvent le code de paiement et le scanner.
- [ ] J’ai testé un petit achat réel.
- [ ] J’ai gardé un autre moyen de paiement prêt.
