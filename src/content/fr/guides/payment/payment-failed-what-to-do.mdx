## Réponse rapide

Si un paiement échoue en Chine, la bonne réaction est simple :

1. faire une pause ;
2. vérifier ce qui s’est réellement passé ;
3. demander si le commerçant a reçu la confirmation ;
4. changer une seule fois de méthode ;
5. avancer.

La mauvaise réaction, c’est de relancer encore et encore sans vérifier.

---

## Étape 1 : savoir ce qui s’est réellement passé

Avant toute chose, il faut déterminer si la situation est :

- clairement échouée ;
- clairement réussie ;
- ou encore en attente.

Si la caisse affiche déjà un succès, ne repayez pas simplement parce que votre application a semblé lente.  
Et si l’application montre une opération en attente, prenez-la au sérieux jusqu’à clarification.

---

## Étape 2 : poser une seule question utile

La meilleure question est souvent :

**“Est-ce que c’est bien passé ?”**

À la caisse, une question courte et calme vaut mieux qu’une longue explication technique.

---

## Étape 3 : changer une seule fois, pas dix

Si le paiement a vraiment échoué, changez une seule fois vers :

- l’autre portefeuille ;
- une autre carte liée ;
- du liquide.

C’est la bonne logique de voyage : faire avancer l’achat sans créer davantage de confusion.

---

## Étape 4 : garder une trace si le statut est flou

Si l’opération paraît en attente ou étrange, gardez :

- une capture d’écran ;
- le montant ;
- l’heure ;
- le nom du commerçant si visible.

Vous réglerez cela plus tard dans un endroit calme, pas pendant qu’une file se forme derrière vous.

---

## Les causes les plus fréquentes

### Refus de la banque

La carte est bien liée, mais la banque bloque la transaction réelle.

### Route commerçant incompatible

Le commerçant accepte le portefeuille, mais pas votre route exacte de carte étrangère.

### Vérification interrompue

L’application demande soudain une validation supplémentaire.

### Retard réseau

Le paiement paraît lent, flou ou brièvement bloqué.

Ces situations sont agaçantes, mais elles ne sont ni rares ni mystérieuses.

---

## Ce qu’il ne faut pas faire

- ne pas appuyer immédiatement une deuxième fois ;
- ne pas supprimer puis rajouter la carte dans la file ;
- ne pas débattre avec le personnel sur la manière dont le système “devrait” fonctionner ;
- ne pas insister par orgueil avec la même méthode qui échoue.

Le but est de terminer l’achat, pas de gagner un débat technique en public.

---

## Quand le liquide est la bonne réponse

Parfois, la décision la plus intelligente consiste simplement à mettre fin au problème.

Le liquide est souvent le bon choix si :

- vous êtes fatigué ;
- la file est longue ;
- le commerçant est petit ;
- le statut du paiement reste flou ;
- vous devez continuer à avancer.

Ce n’est pas une défaite.  
C’est un comportement de voyageur efficace.

---

## Liste de contrôle pratique

- [ ] Je sais vérifier l’historique avant de réessayer.
- [ ] Je pense à demander si le commerçant a reçu la confirmation.
- [ ] Je sais changer une seule fois de méthode, pas sans fin.
- [ ] Je garde des captures si le paiement paraît en attente.
- [ ] Je sais que le liquide peut être la solution la plus rapide.
