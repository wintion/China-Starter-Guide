## Réponse rapide

Si vous comprenez ces deux flux, vous comprenez déjà la plupart des paiements QR en Chine :

- **Le commerçant scanne votre code de paiement**
- **Vous scannez le code du commerçant**

La plupart des confusions initiales proviennent de l'ouverture du mauvais écran, et non de la difficulté du système.

---

## Flux 1 : le commerçant scanne votre code

Ceci est courant dans :

- les supermarchés
- les dépanneurs
- chaînes de restaurants
- des guichets dotés de personnel

Vous ouvrez le code de paiement de votre portefeuille, puis le caissier le scanne.

Il s’agit généralement du flux le plus simple pour les visiteurs car vous n’avez pas besoin de saisir vous-même un montant.

---

## Flux 2 : Vous scannez le code du commerçant

Ceci est courant dans :

- des petits magasins
- quelques cafés
- des stands de rue
- comptoirs libre-service

Vous scannez un code QR commerçant puis confirmez le paiement dans votre application.

Parfois, vous devez saisir le montant manuellement. Si tel est le cas, ralentissez et vérifiez soigneusement avant de payer.

---

## Code marchand officiel vs code de collection personnel

Cette distinction est importante.

En Chine, en particulier dans les petites entreprises, un code de collecte personnel peut parfois vous être présenté au lieu d'une interface marchande plus formelle. Cela ne signifie pas automatiquement qu’il s’agit d’une arnaque. Cela peut être normal pour une très petite entreprise.

Mais les paiements par code personnel sont moins confortables pour les visiteurs car :

- les remboursements sont moins ordonnés
- les factures peuvent être plus difficiles
- vous avez besoin d'être plus sûr que vous payez la bonne personne

En cas de doute, demandez une autre méthode ou payez en espèces.

---

## Les erreurs les plus courantes

- ouvrir le scanner lorsque le caissier a besoin de votre code de paiement
- scanner un code sans vérifier s'il s'agit d'un paiement
- taper le mauvais montant
- réessayer trop rapidement après un certain délai
- paniquer lorsque l'application ouvre une mini-page inconnue

Le remède consiste généralement à faire une pause de cinq secondes et à regarder attentivement.

---

## Une bonne règle pour les petits commerçants

Si le flux de paiement semble un peu compliqué mais que le commerçant est manifestement légitime, restez calme et simple :

- confirmez si vous devez scanner ou présenter votre code
- revérifiez le montant
- payer une fois
- attendre la confirmation

Si quelque chose ne va pas, changez de méthode plutôt que de forcer.

---

## Liste de contrôle pratique

- [ ] Je connais la différence entre « le commerçant me scanne » et « je scanne le commerçant ».
- [ ] Je sais où se trouve mon code de paiement dans chaque application.
- [ ] Je sais qu'il faut revérifier le montant lorsque je le saisis moi-même.
- [ ] Je sais que certains petits commerçants peuvent utiliser des codes de collecte personnels.
- [ ] Je sais changer de méthode si je me sens incertain.
