## Réponse rapide

Vous n’avez pas besoin d’apprendre une longue liste de règles.  
Si vous restez **calme, pratique et observateur**, vous éviterez déjà l’essentiel des frictions.

La plupart des maladresses viennent de trois choses :

- réagir trop fort ;
- expliquer trop longtemps ;
- vouloir absolument faire les choses “à sa manière”.

---

## La règle la plus utile

> **Dans la vie quotidienne en Chine, une coopération calme marche souvent mieux qu’une affirmation trop démonstrative.**

Si vous gardez cela en tête, le reste devient beaucoup plus simple.

---

## Ce qu’il vaut mieux faire

### Être pratique et direct

- dire simplement ce dont vous avez besoin ;
- montrer avec le téléphone plutôt qu’expliquer longtemps ;
- aller vite vers une solution.

Dans beaucoup de contextes, l’efficacité paraît polie.

### Suivre le mouvement

- faire la queue là où les autres font la queue ;
- marcher avec le flux au lieu de le contrarier ;
- observer avant d’agir.

Imiter le rythme local aide énormément.

### Rester calme en public

- garder une voix modérée ;
- contrôler son langage corporel ;
- se mettre un peu de côté si l’on est perdu.

Le calme crée de la confiance.

### Utiliser la technologie

- montrer des adresses en chinois ;
- utiliser les paiements QR ;
- utiliser les applications de traduction.

Le téléphone réduit beaucoup de frictions.

### Accepter le “suffisamment bon”

- les petites imperfections sont normales ;
- une solution approximative mais fonctionnelle est souvent suffisante ;
- avancer compte plus que corriger chaque détail.

Chercher la solution parfaite crée souvent plus de stress qu’elle n’en enlève.

---

## Ce qu’il vaut mieux éviter

### Éviter les disputes publiques

- ne pas transformer un échange avec le personnel en débat ;
- ne pas hausser le ton ;
- ne pas exiger d’explications interminables.

Les disputes publiques rendent en général la situation plus pénible, pas plus efficace.

### Éviter de trop parler

- les longues explications embrouillent ;
- répéter plus fort n’aide pas ;
- les histoires de contexte sont souvent inutiles.

En pratique, plus court marche mieux.

### Ne pas projeter automatiquement les normes occidentales

- le service peut paraître neutre ;
- les sourires ne sont pas garantis ;
- le small talk n’est pas obligatoire.

Neutre ne veut pas dire hostile.

### Ne pas imposer les réflexes de chez soi

- le pourboire n’est généralement pas attendu ;
- une convivialité très expressive peut paraître maladroite ;
- l’espace personnel ne se gère pas toujours comme ailleurs.

S’adapter au rythme local reste la solution la plus fluide.

### Ne pas supposer de mauvaises intentions

- un retard est souvent procédural ;
- une confusion est souvent liée à la langue ;
- un ton neutre est normal.

La plupart des frictions ordinaires ne sont pas personnelles.

---

## Petit pense-bête par situation

### Au restaurant

À faire :

- pointer les plats ;
- payer simplement ;
- partir calmement quand c’est fini.

À éviter :

- laisser un pourboire par réflexe ;
- presser le personnel verbalement ;
- transformer de petits détails en conflit.

### Dans les transports

À faire :

- suivre la signalisation ;
- se ranger sur le côté si l’on est perdu ;
- utiliser les applications pour les taxis ou VTC.

À éviter :

- bloquer les portes ;
- s’arrêter brutalement au milieu du flux ;
- attendre de longues instructions verbales.

### Dans les magasins

À faire :

- vérifier clairement les prix ;
- payer simplement ;
- poser des questions courtes.

À éviter :

- marchander dans les magasins ordinaires ;
- croire que tout se négocie ;
- manipuler les produits sans raison.

### Dans la foule

À faire :

- continuer à avancer ;
- protéger son espace sans agressivité ;
- rester patient.

À éviter :

- pousser franchement ;
- s’arrêter d’un coup ;
- attendre un espace personnel large.

## Pièges fréquents des premiers visiteurs

- croire que parler plus résout tout ;
- interpréter la neutralité comme de l’impolitesse ;
- traiter les différences comme des erreurs ;
- attendre des exceptions partout.

Une fois qu’on voit ces schémas, ils deviennent assez faciles à corriger.

---

## En pratique

- Les habitants savent que vous êtes visiteur.
- L’effort d’adaptation est généralement perçu.
- Un comportement calme inspire du respect.
- Les erreurs sont souvent pardonnées.

On ne vous demande pas d’être parfait sur tous les détails culturels.

---

## Réflexe utile

- rester pratique ;
- éviter le drame ;
- utiliser le téléphone ;
- avancer vite sans s’enfermer.

C’est une habitude simple, locale et très utile à copier.

---

## Liste de contrôle

- [ ] Je reste calme et bref.
- [ ] J’utilise davantage les visuels que les grands discours.
- [ ] Je suis le mouvement local.
- [ ] J’évite les disputes publiques.
- [ ] J’accepte une solution simplement fonctionnelle.
