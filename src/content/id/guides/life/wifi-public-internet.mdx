## Jawaban Singkat

Wi‑Fi publik di Tiongkok ada, tetapi fondasinya lemah untuk perjalanan.

Pakailah sebagai:

- bonus
- backup
- alat untuk tugas yang tidak mendesak

Jangan bangun perjalanan di atasnya untuk:

- pembayaran
- navigasi saat bergerak
- pemulihan login
- tiket atau kode yang sensitif waktu

---

## Default yang Lebih Baik

Anggap **data seluler** sebagai koneksi utama Anda.  
Anggap Wi‑Fi sebagai lapisan kedua.

Perubahan pola pikir sederhana ini menghilangkan banyak stres yang tidak perlu.

---

## Checklist

- [ ] Saya memperlakukan data seluler sebagai koneksi utama.
- [ ] Saya menyimpan alamat dan booking penting secara offline.
- [ ] Saya memakai Wi‑Fi publik hanya untuk tugas non-kritis.
- [ ] Saya tidak mengandalkan Wi‑Fi untuk pembayaran atau verifikasi.
- [ ] Saya punya backup saat koneksi live tidak stabil.
