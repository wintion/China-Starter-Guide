## Jawaban Singkat

Di Tiongkok sekarang, tawar-menawar tidak berlaku di semua tempat.

Masih masuk akal terutama di:

- pasar
- kios suvenir
- pedagang kecil independen

Biasanya tidak cocok di:

- mal
- toko berantai
- restoran
- kafe
- tarif transportasi

Keahlian terpenting bukan menawar keras, tetapi tahu **kapan harga memang sudah tetap**.

---

## Checklist

- [ ] Saya hanya menawar di tempat yang memang informal.
- [ ] Kalau ada label harga jelas, saya anggap itu fixed price.
- [ ] Saya menawar dengan ringan, bukan konfrontatif.
- [ ] Saya siap pergi kalau harga tidak bergerak.
