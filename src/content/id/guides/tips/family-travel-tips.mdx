## Jawaban Singkat

Tiongkok bisa sangat cocok untuk perjalanan keluarga, asalkan Anda merancang hari berdasarkan:

- pace
- jarak jalan
- waktu istirahat

lebih daripada jumlah atraksi.

---

## Checklist

- [ ] Saya tidak memasukkan terlalu banyak tempat dalam satu hari.
- [ ] Saya menghitung waktu istirahat sebagai bagian inti itinerary.
- [ ] Saya memakai mal atau ruang besar sebagai titik reset bila perlu.
- [ ] Saya menghindari hari libur besar jika memungkinkan.
