## Jawaban Singkat

Anda tidak perlu banyak bahasa Inggris lisan untuk tetap bergerak di Tiongkok.

Banyak tugas harian berjalan lewat:

- aplikasi
- layar
- QR
- konfirmasi visual

lebih daripada percakapan panjang.

Kalau ponsel Anda siap, banyak hal terasa jauh lebih mudah.

---

## Checklist

- [ ] Saya memakai layar dan aplikasi sebagai alat utama.
- [ ] Saya menyimpan alamat penting dalam bahasa Mandarin.
- [ ] Saya tidak merasa harus mengobrol panjang untuk menyelesaikan sesuatu.
