## Jawaban Singkat

Navigasi di Tiongkok pada dasarnya lebih **berbasis layar daripada berbasis percakapan**.

Kalau Anda punya:

- aplikasi peta
- alamat Mandarin yang disimpan
- alat terjemahan untuk kalimat singkat

maka Anda biasanya bisa bergerak dengan baik bahkan saat bahasa Inggris lisan sangat terbatas.

---

## Checklist

- [ ] Saya memakai peta sebagai alat utama.
- [ ] Saya menyimpan alamat penting dalam bahasa Mandarin.
- [ ] Saya memakai terjemahan hanya untuk kalimat singkat.
- [ ] Saya siap menunjukkan layar lebih dulu.
